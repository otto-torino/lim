Class: notes

lim. notes

The notes tool class let's the user annotate DOM elements. In edit mode after a click over an element a textarea is prompted to the user so that information for the elemnt can be stored, exisiting information can be edited or deleted. In view mode all the information sotred for DOM elements is presented
